# Student Management System

## Description
A console-based Python project to manage student records efficiently.  
The system allows users to add, view, update, and delete student details.

## Features
- Add student details
- View all student records
- Update student information
- Delete student records

## Technologies Used
- Python

## How to Run
1. Open the student_management_system.py file in Python IDE.
2. Run the program.
3. Follow the on-screen menu options to manage students.
